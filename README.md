# childMaze - 小狗迷宫游戏

一款专为5岁以上儿童设计的迷宫游戏。帮助小狗找到奖品！

## 🎮 游戏说明

### 游戏目标
控制小狗🐕穿越迷宫，找到奖品🏆！

### 操作方式
- **键盘控制**：使用方向键（↑ ↓ ← →）移动小狗
- **触屏/鼠标控制**：点击屏幕上的方向按钮移动小狗
- **过关提示**：按 ENTER 键可关闭过关提示，或等待5秒自动关闭

### 难度等级
- **关卡 1-10**：简单难度，5×5迷宫
- **关卡 11-20**：困难难度，6×6迷宫
- **关卡 21-30**：地狱难度，7×7迷宫

### 游戏特性
- 实时显示当前关卡用时和步数
- 过关后显示鼓励语和统计信息
- 支持触屏设备（已禁用双击放大）
- 响应式设计，适配各种屏幕尺寸
